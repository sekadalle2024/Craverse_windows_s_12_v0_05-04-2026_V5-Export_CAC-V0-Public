┌──────────────────────────────────────────────────────────────────────────────┐
│ 🎯 PROBLEME                                                                  │
└──────────────────────────────────────────────────────────────────────────────┘

  Les 16 sections accordeon des etats de controle etaient affichees
  mais IMPOSSIBLE de les ouvrir en cliquant dessus.

  Cause: onclick="toggleSection(this)" dans le HTML backend
         mais toggleSection() non accessible globalement


┌──────────────────────────────────────────────────────────────────────────────┐
│ ✅ SOLUTION APPLIQUEE                                                        │
└──────────────────────────────────────────────────────────────────────────────┘

  Fichier: public/EtatsControleAccordeonHandler.js
  Ligne 26: window.toggleSection = toggleSection;

  → Expose la fonction globalement
  → Compatible avec onclick inline du HTML backend
  → Une seule ligne ajoutee!
