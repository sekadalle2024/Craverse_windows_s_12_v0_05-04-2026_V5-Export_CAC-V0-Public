COMMENT UTILISER

MÉTHODE 1 : Menu Contextuel
----------------------------
1. Taper "Etat fin" dans le chat
2. Sélectionner fichier Balance Excel
3. Clic droit → "📥 Exporter liasse officielle"
4. Ouvrir le fichier Excel
5. Premier onglet = "Contrôle de cohérence"

MÉTHODE 2 : Script Python
--------------------------
cd py_backend
conda activate claraverse_backend
python generer_etats_liasse.py

→ Fichier HTML sur le Bureau
→ Export Excel avec onglet "Contrôle de cohérence"

MÉTHODE 3 : API
---------------
POST /export-liasse/generer
Body: {"results": {...}, "nom_entreprise": "...", "exercice": "2024"}

→ Fichier Excel en base64
→ Onglet "Contrôle de cohérence" inclus
