🎯 LES 3 PROBLÈMES EN DÉTAIL
================================================================================

PROBLÈME 1: Colonnes BRUT et AMORTISSEMENT
   Symptôme:
   - Le template Excel a 3 colonnes pour N: BRUT, AMORTISSEMENT, NET
   - Seule la colonne NET est renseignée
   - Le menu accordéon n'affiche que NET
   
   Solution:
   - Extraire les valeurs brutes depuis les comptes 2xxx
   - Extraire les amortissements depuis les comptes 28xx
   - Calculer: NET = BRUT - AMORTISSEMENT
   - Remplir les 3 colonnes dans l'export
   - Ajouter les colonnes dans le menu accordéon

PROBLÈME 2: Totalisations manquantes
   Symptôme:
   - Les lignes de totalisation restent vides:
     * AZ: TOTAL ACTIF IMMOBILISÉ
     * BQ: TOTAL ACTIF CIRCULANT
     * BZ: TOTAL TRÉSORERIE-ACTIF
     * CZ: TOTAL ÉCARTS DE CONVERSION-ACTIF
     * DZ: TOTAL GÉNÉRAL ACTIF
   
   Solution:
   - Identifier les postes inclus dans chaque totalisation
   - Calculer les sommes (brut, amortissement, net)
   - Remplir les lignes de totalisation

PROBLÈME 3: TFT vierge
   Symptôme:
   - L'onglet TFT reste complètement vide
   - Le TFT est pourtant généré (format dict)
   
   Solution:
   - Vérifier le mapping des colonnes (I et K au lieu de H et I)
   - Améliorer la conversion dict → liste de postes
   - Scanner correctement la colonne A pour les REF
