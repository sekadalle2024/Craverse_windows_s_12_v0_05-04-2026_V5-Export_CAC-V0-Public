ÉTAPE 2: Identification du problème
------------------------------------
PROBLÈME TROUVÉ dans py_backend/etats_financiers.py (lignes 1742-1747):

Code incorrect:
```python
# Séparer les données N et N-1 pour les états de contrôle
bilan_actif_n = [p for p in results_liasse['bilan_actif']]
bilan_actif_n1 = [p for p in results_liasse['bilan_actif']]  # ❌ MÊME DONNÉES!
```

CAUSE:
• Les données N et N-1 ne sont pas séparées correctement
• bilan_actif_n et bilan_actif_n1 pointent vers les mêmes données
• Les états de contrôle utilisent donc les mêmes valeurs pour N et N-1

IMPACT:
• Les colonnes N-1 ne sont pas affichées correctement
• Les états de contrôle calculent des différences nulles
• Le TFT peut être affecté si les données N-1 sont incorrectes

EXPLICATION:
• Les données sont déjà au format liasse avec montant_n, montant_n1 et montant_n2

• Les fonctions de contrôle (etats_controle_exhaustifs.py) lisent directement montant_n et montant_n1
