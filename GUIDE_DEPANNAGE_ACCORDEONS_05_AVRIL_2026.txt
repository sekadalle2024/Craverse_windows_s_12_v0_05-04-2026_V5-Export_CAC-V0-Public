ETAPE 1: TEST STANDALONE
=========================

1. Ouvrir le fichier: test-accordeons-standalone.html
2. Cliquer sur les 3 sections
3. Verifier qu'elles s'ouvrent et se ferment

SI CA FONCTIONNE:
→ Le code JavaScript est correct
→ Le probleme vient de l'integration dans l'application

SI CA NE FONCTIONNE PAS:
→ Probleme avec le navigateur ou les styles CSS
→ Essayer un autre navigateur

2. Ouvrir la console (F12)
3. Generer les etats: "Etat fin" + upload balance

=============================

Dans la console (F12), executer:

// Attendu: true (si ouverte)

ETAPE 7: SOLUTION MANUELLE
===========================

Si rien ne fonctionne, executer ce code dans la console:

// Solution de secours
document.querySelectorAll('.section-header').forEach((header, index) => {
    // Supprimer onclick
    header.removeAttribute('onclick');
    
    // Ajouter listener
    header.addEventListener('click', function() {
        const section = this.parentElement;
        section.classList.toggle('active');
        console.log(`Section ${index + 1} toggled:`, section.classList.contains('active'));
    });
    
    console.log(`Listener manuel ajouté sur section ${index + 1}`);
});

console.log('✅ Listeners manuels ajoutés!');


PROBLEMES FREQUENTS
===================

PROBLEME 1: "Nombre de sections trouvées: 0"
CAUSE: Les sections ne sont pas encore dans le DOM
SOLUTION: Attendre ou reinitialiser
COMMANDE: window.reinitializeEtatsControleAccordeons();

PROBLEME 2: Clic ne fait rien
CAUSE: Event listeners non attaches
SOLUTION: Verifier les logs "✅ Listener ajouté"
COMMANDE: Voir ETAPE 7 (Solution manuelle)

PROBLEME 3: Section ne s'ouvre pas visuellement
CAUSE: Styles CSS manquants
SOLUTION: Verifier max-height dans les styles
FICHIER: py_backend/etats_financiers.py

PROBLEME 4: Erreur "toggleSection is not defined"
CAUSE: Script non charge ou fonction non exposee
SOLUTION: Verifier index.html ligne 151
FICHIER: index.html

PROBLEME 5: Conflits avec attributs onclick
CAUSE: Attributs onclick du HTML backend
SOLUTION: Les attributs sont maintenant supprimes automatiquement
VERIFICATION: Logs "🔧 Attribut onclick supprimé"
