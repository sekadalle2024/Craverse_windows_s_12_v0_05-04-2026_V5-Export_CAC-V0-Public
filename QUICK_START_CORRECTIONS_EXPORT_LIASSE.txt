================================================================================
ÉTAPES DE TEST
================================================================================

1️⃣  DÉMARRER LE BACKEND
   
   Commande:
   .\start-claraverse.ps1
   
   Attendre le message: "✅ Backend démarré sur http://localhost:8000"

2️⃣  IMPORTER LA BALANCE DE TEST
   
   Fichier: py_backend/P000 -BALANCE DEMO N_N-1_N-2.xls
   
   Dans l'interface:
   - Cliquer sur "Importer Balance"
   - Sélectionner le fichier
   - Attendre la confirmation

3️⃣  GÉNÉRER LES ÉTATS FINANCIERS
   
   Dans l'interface:
   - Cliquer sur "Générer États Financiers"
   - Attendre la génération (quelques secondes)
   - Vérifier que les états s'affichent

4️⃣  EXPORTER LA LIASSE OFFICIELLE
   
   Dans l'interface:
   - Cliquer sur "Exporter Liasse Officielle"
   - Le fichier Excel se télécharge automatiquement
   - Nom: Liasse_Officielle_[Entreprise]_[Année].xlsx

5️⃣  VÉRIFIER LE FICHIER EXCEL
   
   Ouvrir le fichier téléchargé et vérifier:
   
   📊 ONGLET ACTIF:
   ✓ Colonne F (BRUT) remplie pour AD, AE, AF, AG, AH, AI, AJ, AK, AL, AM, AN, AP, AQ
   ✓ Colonne G (AMORTISSEMENT) remplie
   ✓ Colonne H (NET N) remplie
   ✓ Colonne I (NET N-1) remplie
   ✓ Cohérence: H ≈ F - G (à ±1 près)
   
   📊 TOTALISATIONS:
   ✓ Ligne AZ (TOTAL ACTIF IMMOBILISÉ) avec F, G, H, I
   ✓ Ligne BQ (TOTAL ACTIF CIRCULANT) avec H, I
   ✓ Ligne BZ (TOTAL TRÉSORERIE-ACTIF) avec H, I
   ✓ Ligne CZ (TOTAL ÉCARTS DE CONVERSION-ACTIF) avec H, I
   ✓ Ligne DZ (TOTAL GÉNÉRAL ACTIF) avec F, G, H, I
   
   📊 ONGLET TFT:
   ✓ Colonne I (N) remplie
   ✓ Colonne K (N-1) remplie

     → Réimporter la balance
     → Réessayer
