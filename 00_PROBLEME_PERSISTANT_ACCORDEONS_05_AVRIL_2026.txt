PROCHAINES ETAPES
=================

ETAPE 1: TEST STANDALONE
-------------------------
1. Ouvrir: test-accordeons-standalone.html
2. Cliquer sur les sections
3. Verifier qu'elles s'ouvrent

→ Si ca fonctionne: Le code est correct, probleme d'integration
→ Si ca ne fonctionne pas: Probleme de navigateur ou CSS


ETAPE 2: DIAGNOSTIC DANS L'APPLICATION
---------------------------------------
1. Demarrer: npm run dev
2. Ouvrir console (F12)
3. Generer: "Etat fin" + upload balance
4. Chercher les logs:
   - "🔧 EtatsControleAccordeonHandler: Initialisation..."
   - "📊 Nombre de sections trouvées: X"
   - "✅ Listener ajouté sur section X"

→ Si "Nombre de sections trouvées: 0":
   Executer: window.reinitializeEtatsControleAccordeons();


ETAPE 3: TESTS MANUELS
-----------------------
Dans la console (F12):

// Test 1: Compter les sections
document.querySelectorAll('.section-header').length

// Test 2: Tester toggleSection
window.toggleSection(document.querySelector('.section-header'));

// Test 3: Verifier les styles
const content = document.querySelector('.section-content');
console.log(window.getComputedStyle(content).maxHeight);


ETAPE 4: SOLUTION DE SECOURS
-----------------------------

HYPOTHESES
==========

HYPOTHESE 1: Les sections ne sont pas dans le DOM
→ Test: document.querySelectorAll('.section-header').length
→ Si 0: Reinitialiser avec window.reinitializeEtatsControleAccordeons();

HYPOTHESE 2: Les styles CSS ne sont pas charges
→ Test: window.getComputedStyle(document.querySelector('.section-content')).maxHeight
→ Si undefined: Verifier py_backend/etats_financiers.py

HYPOTHESE 3: Conflit avec d'autres scripts
→ Test: Chercher erreurs JavaScript dans la console
→ Si erreurs: Verifier l'ordre de chargement dans index.html

HYPOTHESE 4: Le HTML est genere apres l'initialisation
→ Test: Chercher logs "🆕 Nouvelles sections détectées"
→ Si absent: Le MutationObserver ne detecte pas les changements

HYPOTHESE 5: Les event listeners ne sont pas attaches
→ Test: document.querySelector('.section-header').dataset.listenerAttached
→ Si undefined: Les listeners ne sont pas attaches
