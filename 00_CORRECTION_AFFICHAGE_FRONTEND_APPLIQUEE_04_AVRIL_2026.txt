    }
  }
});

STRUCTURE DES ACCORDÉONS
═══════════════════════════════════════════════════════════════════════════════

Les accordéons affichés dans le chat :

1. 📊 BILAN ACTIF (ouvert par défaut)
   - Colonnes : REF | LIBELLÉS | NOTE | EXERCICE N | EXERCICE N-1
   - Tous les postes de l'actif (AA à DZ)

2. 📊 BILAN PASSIF
   - Colonnes : REF | LIBELLÉS | NOTE | EXERCICE N | EXERCICE N-1
   - Tous les postes du passif (CA à DZ)

3. 📊 COMPTE DE RÉSULTAT
   - Colonnes : REF | LIBELLÉS | NOTE | EXERCICE N | EXERCICE N-1
   - Tous les postes du compte de résultat (TA à XN)

4. 📊 TABLEAU DES FLUX DE TRÉSORERIE (TFT)
   - Colonnes : REF | LIBELLÉS | NOTE | EXERCICE N | EXERCICE N-1
   - Flux de trésorerie d'exploitation, d'investissement et de financement

5. 📊 NOTES ANNEXES
   - Informations complémentaires sur les états financiers

═══════════════════════════════════════════════════════════════════════════════
RAPPEL : FORMAT D'AFFICHAGE
═══════════════════════════════════════════════════════════════════════════════

⚠️ IMPORTANT :
- N-2 est utilisé pour les CALCULS (variations de N-1)
- N-2 n'est PAS affiché comme colonne séparée
- Seules 2 colonnes sont affichées : EXERCICE N et EXERCICE N-1
- Format conforme à la liasse officielle SYSCOHADA

EXEMPLE D'AFFICHAGE :
┌──────┬─────────────────────────────────┬──────┬──────────────┬──────────────┐
│ REF  │ LIBELLÉS                        │ NOTE │ EXERCICE N   │ EXERCICE N-1 │
├──────┼─────────────────────────────────┼──────┼──────────────┼──────────────┤
│ TA   │ Ventes de marchandises          │ 21   │ 1 000 000    │ 950 000      │
│ RA   │ Achats de marchandises          │ 22   │ 600 000      │ 570 000      │
│ RB   │ Variation de stocks             │ 23   │ 50 000       │ 45 000       │
│ XA   │ MARGE COMMERCIALE (TA-RA-RB)    │      │ 350 000      │ 335 000      │
└──────┴─────────────────────────────────┴──────┴──────────────┴──────────────┘
