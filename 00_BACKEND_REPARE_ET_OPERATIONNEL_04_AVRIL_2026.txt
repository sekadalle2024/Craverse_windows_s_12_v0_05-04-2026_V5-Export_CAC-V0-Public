WORKFLOW COMPLET "ETAT FIN":
----------------------------
1. Frontend détecte "Etat fin" dans le message
2. claraApiService.ts génère une table déclencheuse
3. EtatFinAutoTrigger.js détecte la table
4. Dialogue de fichier s'ouvre automatiquement
5. Utilisateur sélectionne le fichier Excel de balance
6. Fichier envoyé au backend via /etats-financiers/process-excel
7. Backend traite les 3 onglets (N, N-1, N-2)
8. Backend calcule les états financiers SYSCOHADA
9. Backend génère le HTML avec accordéons
10. Frontend reçoit le HTML
11. EtatFinAutoTrigger.js remplace la table par le HTML
12. Accordéons affichés et fonctionnels
